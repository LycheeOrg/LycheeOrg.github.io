The current API provide the following entry points: See:

  • routes/web.php
  • routes/admin.php

php/index.php, /api/Session::init

  • POST request
  • 0 arguments
  • no CSRF protection
  • Returns the server informations, including the locale translations.

/api/Session::login

  • POST request
  • Returns true if login was successful, false otherwise.
$request->validate([
    'user' => 'required',
    'password' => 'required',
]);

/api/Session::logout

  • POST request
  • 0 arguments
  • Returns true and log out (flush Session variables)
