The Timeline provides a chronological view of your photos, making it easy to browse your library by date.

## Features

- **Standalone page** — access the timeline as a dedicated page
- **Direct date access** — jump to a specific date
- **Access control** — respects album-level permissions

## Supporter Edition Features

With the [Supporter Edition](/docs/se/overview/):

- **Custom granularity** — choose the time grouping (day, week, month, year)
- **Custom format** — customize the date display format
- **Customizable timeline per album** — configure timeline settings on a per-album basis
