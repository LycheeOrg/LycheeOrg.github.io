Lychee includes a built-in slideshow mode for viewing photos in fullscreen with automatic transitions.

## Features

- Fullscreen photo display
- Previous and next navigation
- Auto-advance with configurable timing
- Works within albums and across the library
- Keyboard navigation support (see [Keyboard Shortcuts](/docs/usage/keyboard/))
