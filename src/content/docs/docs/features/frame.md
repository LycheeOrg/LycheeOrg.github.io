The Frame module turns Lychee into a digital photo frame, cycling through your photos in a fullscreen display.

## Features

- Fullscreen photo display mode
- Automatic photo rotation
- Configurable display interval
- Works with any album or the entire library

## Use Cases

- Digital photo frame displays on dedicated screens
- Waiting room or reception area photo slideshows
- Gallery exhibition displays
