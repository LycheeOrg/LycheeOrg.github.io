The Map module displays your geotagged photos on an interactive map, showing where your photos were taken.

## Features

- Interactive map with photo markers
- Cluster markers for areas with many photos
- Click markers to preview photos
- Filter by album or date range

## Requirements

Photos must contain GPS data in their EXIF metadata for map placement. Most modern smartphones and GPS-enabled cameras embed this data automatically.

## Configuration

The map module can be enabled or disabled in the [Settings](/docs/getting-started/settings/). You can also configure the map provider and default zoom level.
