Flow displays albums in a feed-like manner, similar to social media platforms. It provides a scrollable, visually engaging way to browse your photo library.

## Features

- **Standalone page** — access Flow as a dedicated page
- **Access control** — respects album-level permissions
- **Configurable base album** — choose which album tree feeds the Flow

## Supporter Edition Features

With the [Supporter Edition](/docs/se/overview/):

- **Compact mode** — display flow cards in a more compact layout
- **Extensive customizations** — fine-tune the appearance of flow cards
- **Custom strategy** — choose between opt-in and automatic inclusion of albums (coming soon)
