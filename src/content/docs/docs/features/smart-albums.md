Smart Albums are dynamic albums that automatically collect photos based on specific criteria. They update in real-time as your library changes.

## Built-in Smart Albums

Lychee provides several built-in smart albums:

- **Recent** — recently added photos
- **Starred** — photos you've marked as favorites
- **On This Day** — photos taken on today's date in previous years
- **Unsorted** — photos not assigned to any album

## Per-User Smart Albums

Each user gets their own set of smart albums, showing only photos they have access to.

## Custom Smart Albums

Create smart albums based on custom tag selections to automatically group photos by topic, event, or any criteria you define.

See also [Tagging](/docs/features/tagging/) for how to organize photos with tags.
