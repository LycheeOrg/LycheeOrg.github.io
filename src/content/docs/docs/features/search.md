Lychee provides search functionality to quickly find photos and albums across your library.

## Search Capabilities

- Search by photo title and description
- Search by album name
- Search by tags
- Search across your entire accessible library

## Public Search

Administrators can enable or disable public search to allow unauthenticated visitors to search through public albums.
