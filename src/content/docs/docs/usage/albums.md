Albums are the primary way to organize photos in Lychee. You can create unlimited albums, nest them within each other, and customize their appearance.

## Creating Albums

To create an album, click the **+** button in the top bar and select **New Album**. You can also create albums within albums for hierarchical organization.

## Album Features

- **Unlimited albums** — no restrictions on the number of albums
- **Albums within albums** — organize your library with nested album structures
- **Customizable covers** — choose which photo represents the album
- **Customizable header photo** — set a banner image at the top of the album
- **Customizable layouts** — choose between justified, masonry, or square photo layouts
- **Customizable sorting** — sort photos by date, title, or other criteria
- **Customizable aspect ratio** — control thumbnail proportions in album views
- **Timeline view** — browse photos chronologically within an album
- **Embed support** — embed albums in other websites

## Album Sharing

Albums can be shared with specific users or made public. See [Sharing](/docs/usage/sharing/) for details on access control.

## Downloading Albums

Albums can be downloaded as ZIP archives containing all photos at their original resolution.

## Per-Album Customizations

Each album supports individual customization of:

- Photo sorting order
- Album cover selection
- Header photo
- Thumbnail aspect ratio
- Photo layout style (justified, masonry, square)
- Timeline display (Supporter Edition)
