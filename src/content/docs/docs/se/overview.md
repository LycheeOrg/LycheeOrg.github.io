The **Lychee Supporter Edition (SE)** includes all features of the Free Edition, plus exclusive additions as a thank-you for supporting the development of Lychee.

## How to Get the Supporter Edition

1. Support Lychee via [GitHub Sponsors](https://github.com/sponsors/LycheeOrg).
2. Retrieve your personal license key from [keygen.lycheeorg.dev](https://keygen.lycheeorg.dev)
3. In Lychee, open the **Left menu** → **About Lychee** dialog → enter your license key

## Exclusive Features

| Feature | Description |
|---------|-------------|
| [Statistics](/docs/se/statistics/) | Detailed statistics on your library, albums, and usage |
| [User Groups](/docs/se/user-groups/) | Organize users into groups with shared permissions |
| [Watermarking](/docs/se/watermarking/) | Apply custom watermarks to your photos |
| [AES-Secured Links](/docs/se/aes-links/) | Encrypted, tamper-proof photo sharing links |
| [White Label](/docs/se/white-label/) | Remove all Lychee branding for a custom look |
| [Webshop](/docs/webshop/overview/) | Sell your photos with integrated payment support (Pro) |
| Custom timeline per album | Configure timeline granularity and format per album |
| Flow customizations | Compact mode and extensive flow card customizations |
| Tag display on albums | Show tags directly in album views |
| Upload quotas | Set per-user upload limits |
| Multi-admin support | Delegate admin responsibilities |
| Contact form | Built-in contact form module |
| Personal favourites | User-specific favourite collections |
| Renaming rules | Custom file renaming on import/upload |

## Licensing

The Supporter Edition uses the same **MIT license** as the Free Edition. Lychee remains fully open source — the SE code is available on GitHub.

## Managing Your Subscription

You can manage your GitHub Sponsors subscription directly on [GitHub](https://docs.github.com/en/billing/managing-billing-for-your-products/managing-billing-for-github-sponsors/downgrading-a-sponsorship).

## Lost Your License Key?

Retrieve a new key at [keygen.lycheeorg.dev](https://keygen.lycheeorg.dev).
