The White Label option allows you to remove all mentions of Lychee from the user interface, presenting a fully branded experience to your users.

## Features

- Remove all Lychee branding and logos
- Present a clean, custom-branded photo gallery
- Ideal for professional photographers and businesses

## Configuration

Enable the white label option in the Lychee settings panel. Once active, all Lychee branding is hidden from the public-facing interface.
